PoshZone Demo

Please install this version of operator into Firefox for this demo. I haven't been able to set the mime type of the file on the server so you will have to save it to disk and then open it in Firefox to install it as a plugin. Next you will need to register poshZone as a dataformat and then add some sample poshZone actions into operator.
More detailed steps here
The content below is marked up using poshZones. Once you have restarted FireFox and reloaded this page you should see poshZone(4) highlighted in your operator toolbar.
To see the results of an action on a new tab, hold ctrl whilst clicking the action.
